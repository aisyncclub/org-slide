# 프리셋: editorial — 에디토리얼 빅타입 (Editorial Bold)

> **스타일 정체성:** 출판물·매거진 감성. **거대한 세리프 헤드라인**이 주인공.
> 잉크 1색 + 강조 1색(버밀리언), 넓은 여백·비대칭 그리드·풀쿼트.
> (2026 트렌드: 빅타입 에디토리얼 — 헤드라인이 화면의 주연)

## 톤 키워드
**묵직 · 출판물 · 사색 · 권위 · 스토리**

아이보리 베이스 + 진한 잉크 + 버밀리언 강조. 큰 세리프 제목으로 메시지 하나를 강하게.
장식 최소화, 타이포그래피와 여백으로 승부.

## 컬러 팔레트

| 이름 | HEX | 용도 |
|---|---|---|
| 아이보리 | `#FBF9F4` | 베이스 |
| 잉크 블랙 | `#161616` | 헤드라인·본문 (메인) |
| 버밀리언 | `#E63946` | 강조 1색 |
| 오커 골드 | `#C9A227` | 2차 강조(절제) |

## :root 토큰

```css
:root{
  /* 면 */
  --bg:        #FBF9F4;   /* 아이보리 */
  --surface:   #FFFFFF;
  --surface-2: #F3EFE6;
  /* 잉크 */
  --ink:       #161616;
  --ink-soft:  #4A4642;
  --ink-dim:   #8A847A;
  /* 브랜드 */
  --brand:     #161616;   /* 잉크(메인 — 큰 세리프 제목) */
  --brand-2:   #4A4642;
  --accent:    #E63946;   /* 버밀리언 (강조 1색) */
  --accent-2:  #C9A227;   /* 오커 골드 (2차) */
  /* 라인/하이라이트 */
  --rule:      #E3DDD0;
  --hl:        rgba(230,57,70,0.16);   /* 버밀리언 형광펜 */
  /* 타이포 */
  --display:   "Fraunces","Noto Serif KR",Georgia,serif;   /* 세리프 디스플레이 */
  --sans:      "Pretendard Variable",Pretendard,"Noto Sans KR",sans-serif;
  --mono:      "JetBrains Mono",ui-monospace,monospace;
  --pad: 64px 88px;
}
```

CDN 폰트(세리프 디스플레이) 추가:
```html
<link href="https://fonts.googleapis.com/css2?family=Fraunces:opsz,wght@9..144,600;9..144,700&family=Noto+Serif+KR:wght@600;700;900&display=swap" rel="stylesheet">
```

## 폰트 규칙
- 제목: **Fraunces 700 / Noto Serif KR 900** — 아주 크게(표지 96–140px). italic 허용(세리프 강조).
- 본문/캡션: **Pretendard 400/500** (세리프 제목과 대비).
- 풀쿼트: 세리프 + 버밀리언 인용부호.

## 레이아웃 시그널
- **빅타입 표지**: 화면의 50%+를 차지하는 세리프 헤드라인. 좌측 정렬, 넓은 여백.
- 얇은 잉크 룰(상단/하단 호리즌 라인), 페이지 번호 세리프.
- 비대칭 2단 그리드(텍스트 70 / 여백·캡션 30).
- 강조는 버밀리언 형광펜 또는 버밀리언 드롭캡. 면적은 작게.
- 차트도 잉크+버밀리언 단색 위주(에디토리얼 절제).

## 차트 팔레트
`#161616` · `#E63946` · `#C9A227` · `#8A847A` (잉크 중심 + 버밀리언 강조)
text=`#4A4642`, grid=`#E3DDD0`.

## 모션
정적·우아하게. `opacity 0→1 + 약한 y 12px`, 헤드라인은 글자 라인 단위 살짝 stagger.

## 금지
- 다크 배경, 네온, 과한 컬러. 버밀리언 외 무지개색.
- 작은 제목(빅타입 정체성 훼손). 표지 헤드라인은 반드시 크게.
- **특정 기관/조직명 하드코딩** — `[조직명]` 플레이스홀더.
